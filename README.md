## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone repository
git clone https://github.com/fitriadamayanti12/personal-website.git

# Navigate to project
cd personal-website

# Install dependencies
npm install

# Run development server
npm run dev
